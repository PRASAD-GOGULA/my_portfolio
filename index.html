Projects

AXI Protocol

Developing an AXI-compliant interface involves designing synthesizable RTL modules in Verilog to enable high-speed, efficient communication between processors, memory, and peripherals in ARM-based systems, with careful implementation of all AXI protocol channels—including read and write address, write data, read data, and response paths—while ensuring proper handshaking, burst transactions, and timing requirements are met; the design is then rigorously validated through simulation to verify protocol compliance, data integrity, and correct behavior under various traffic and corner-case scenarios.

APB Protocol

Designed and implemented the AMBA 3 APB protocol in RTL using Verilog, covering all key signals and ensuring correct state transitions between setup and access phases, and further verified the implementation using UVM-based testbenches to validate read and write transactions, as well as to ensure proper protocol behavior and state transition accuracy under different scenarios.

Dual-Port Ram

Designed and implemented a dual-port RAM in RTL using Verilog, enabling simultaneous read and write operations on independent ports with proper handling of address, data, and control signals, and developed a comprehensive testbench to verify concurrent access, read/write functionality, and edge cases, while leveraging waveform debugging to analyze signal activity, identify timing issues, and ensure correct and reliable simulation behavior.
